Dynamic typing

JavaScript is a loosely typed or a dynamic language. Variables in JavaScript are not directly associated with any particular value type, and any variable can be assigned (and re-assigned) values of all types

                                    let foo = 42;// foo is now a number
                                    foo = 'bar'// foo is now a string
                                    foo = true// foo is now a boolean
Two Kinds of Data

The latest ECMAScript standard defines nine types:
In JavaScript there are two different kinds of data: primitives , and objects. A primitive is simply a data type that is not an object, and has no methods.
Primitive values: strings, numbers, booleans – are assigned/copied “as a whole value”.

  • A boolean represents only one of two values: true, or false. Think of a boolean as an on/off or a yes/no switch. 

                                                    var bool = true;
                                                var boo2 = false;

  • There is only one type of Number in JavaScript. Numbers can be written with or without a decimal point. A number can also be +Infinity, -Infinity, and NaN (not a number). 

                                                    var num1 = 32;
                                                var num2 = +Infinity;

  • Strings are used for storing text. Strings must be inside of either double or single quotes. 

                                                    var str1 = 'hello, it is me';
                                                var str2 = 'hello, it is you';

  • Null has one value: null. It is explicitly nothing. 

                                                    var nothing = null;

  • A variable that has no value is undefined. 

                                                var testVar;
                                            console.log(testVar);// undefined

  • Symbols are new in ES6. A Symbol is an immutable primitive value that is unique. 

                                            const mySymbol = Symbol('mySymbol');

  • BigInt is a special numeric type that provides support for integers of arbitrary length.
    A bigint is created by appending n to the end of an integer literal or by calling the function BigInt that creates bigints from strings, numbers etc. 

                                        const bigint = 1234567890123456789012345678901234567890n;

What about Objects?

Objects are not a primitive data Type.
An object is a collection of properties. These properties are stored in key/value pairs. Properties can reference any type of data, including objects and/or primitive values.
One of the fundamental differences of objects vs primitives is that they are stored and copied “by reference”. 

                                
                                var obj = {
                                    key1: 'value',
                                    key2: 'value',
                                    key3: true,
                                    key4: 32,
                                    key5: {}
                                  }

A variable stores not the object itself, but its “address in memory”, in other words “a reference” to it. When an object variable is copied – the reference is copied, the object is not duplicated.

Loosely Typed

JavaScript is a loosely typed language. This means you don’t have to declare a variable’s type. JavaScript automatically determines it for you. It also means that a variables type can change.
